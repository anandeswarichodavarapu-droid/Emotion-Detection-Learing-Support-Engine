# Emotion Detection & Learning Support Engine

An AI-powered educational support platform that detects learner emotions using BiLSTM and BERT, then generates personalized learning guidance using Google Gemini AI.

## Features

- Emotion Detection
- Mixed Emotion Analysis
- BiLSTM Model
- BERT Model
- Gemini AI Guidance
- Plotly Analytics
- CSV Logging
- Session History
- Streamlit Dashboard

## Tech Stack

- Python
- Streamlit
- TensorFlow
- PyTorch
- Transformers
- Google Gemini AI
- Plotly
- Pandas
